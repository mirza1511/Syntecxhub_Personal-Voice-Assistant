🧠 SyntecxHub Assistant

A simple Python-based terminal personal assistant that performs basic tasks like showing time, date, opening websites, and telling jokes — without any external libraries.

🚀 Features
👋 Greets user based on time
⏰ Shows current time
📅 Shows current date
🌐 Opens websites (Google, YouTube, GitHub)
😂 Tells random programming jokes
❓ Help command for all features
📴 Exit command to stop program

▶️ How to Run
python "Project 1.py"

⚠️ If possible, rename file to project1.py for easier use.

💡 Commands
hello
time
date
open google
open youtube
open github
joke
help
exit
